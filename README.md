# Web Development Mini Projects

A collection of responsive front-end web pages built using **HTML** and **CSS** to practice modern UI layout and styling.

## Projects Included
- **Forever21 Landing Page Clone**  
  A simple e-commerce landing page UI with navigation, search bar, and banner section.

- **YouTube Homepage Clone**  
  A YouTube-style homepage layout featuring a fixed header, sidebar navigation, and video grid.

## Tech Stack
- HTML5  
- CSS3  

## How to Run
1. Download or clone this repository
2. Open any `.html` file in your browser:
   - `FOREVER21.html`
   - `YouTube.html`
